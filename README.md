# Nutriary
Daily Nutrition and Calorie Diary Android App 
**Features**
```
- BMR Calculator
- Entering Calorie Intake by Searching the food and inputing the quantity of the food
- Total Daily Nutrition (Calorie, Protein, Natrium, Sugar, Fats, etc)
```
